Wickedpicker

wickedpicker.js - A simple jQuery timepicker

Requirements

* [jQuery](http://jquery.com/download/) (>= 1.9)

Usage

In your HTML
In your JavaScript file 
            $('.timepicker').wickedpicker();
Optional twenty-four hour clock setting (default false) 
            $('.timepicker').wickedpicker({twentyFour: true});
Methods 'time' get the current time inside of the input element that has a wickedpicker attached to it. 
            $('.timepicker').wickedpicker('time');
If multiple input fields have the same class and instantiate a wickedpicker then pass the index of the timepicker you'd like to select 
            $('.timepicker').wickedpicker('time', 0);
